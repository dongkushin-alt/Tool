# Quotation Tool (견적서 자동 생성 툴)

## 📋 개요
Excel VBA 기반의 견적서 자동 생성 도구입니다. 
부품 데이터베이스에서 부품번호와 수량을 선택하면, 자동으로 부품명, 원가, 판매가격을 계산하여 견적서를 생성합니다.

## ✨ 주요 기능
- ✅ 부품번호 & 수량 입력 → 자동 부품명/가격 조회
- ✅ 마진율 적용 → 판매가격 자동 계산
- ✅ 견적서 자동 생성 (Excel 형식)
- ✅ 부품 DB 관리 (카테고리별 관리)
- ✅ 마진율 설정 (카테고리별/전체 설정)

## 🚀 사용 방법

### 1. 초기 설정
1. `Quotation_Tool.xlsm` 파일을 다운로드하여 Excel에서 열기
2. "부품DB" 시트에 부품 정보 입력 (또는 기존 데이터 import)
3. "마진설정" 시트에서 마진율 설정

### 2. 견적서 생성
1. "견적서 작성" 시트 열기
2. 부품번호 입력 → 자동으로 부품명/원가 조회
3. 수량 입력
4. "견적서 생성" 버튼 클릭
5. 자동으로 판매가격 계산 및 견적서 생성

### 3. 견적서 다운로드
- 생성된 견적서는 자동으로 새 Excel 파일로 저장됨
- 필요에 따라 수정 후 PDF로 내보내기

## 📊 부품 DB 칼럼 구조

| 부품번호 | 부품명 | 카테고리 | 원가 | 단위 | 기타정보 |
|---------|--------|---------|------|------|----------|
| P001 | 베어링 | 기계부품 | 5,000 | 개 | - |
| P002 | 모터 | 전기부품 | 25,000 | 개 | 2HP |

## 💰 마진 설정

- **전체 기본 마진**: 20%
- **카테고리별 마진**: 기계부품(15%), 전기부품(25%), 재료(30%)
- 동적으로 수정 가능

## 🛠 VBA 매크로 구성

| 매크로명 | 기능 |
|---------|------|
| SearchPart() | 부품번호로 부품 정보 조회 |
| CalculatePrice() | 원가 + 마진 = 판매가 계산 |
| GenerateQuote() | 견적서 생성 및 저장 |
| ValidateInput() | 입력값 검증 |

## ⚙️ 시스템 요구사항
- Windows 10 이상 또는 Mac (Excel 2016 이상)
- Microsoft Excel (VBA 매크로 지원)
- 마크로 실행 허용 설정

## 📝 주의사항
- 매크로 실행 시 보안 경고가 나타날 수 있으니 "매크로 활성화" 클릭
- 부품 DB는 정기적으로 백업 권장
- 견적서는 자동으로 'Quotations' 폴더에 저장됨
