## 📋 2단계: Quotation_Tool.xlsm 파일 만들기

### 2-1. Excel 파일 생성 및 저장

```
1. Microsoft Excel 열기
   ↓

   ↓

   - 저장 위치: 원하는 폴더
   ↓
4. 저장 완료
```

> **중요:** 반드시 .xlsm 형식으로 저장하세요!

> ⚠️ 시트명을 정확하게 입력하세요! (공백 포함)

## 💻 4단계: VBA 코드 입력

### 4-1. VBA 에디터 열기
```
Excel 파일에서 Alt + F11
↓
VBA 에디터 창 열림
```

### 4-2. 코드 입력

**코드 1: 메인_매크로**
1. GitHub에서 `메인_매크로.bas` 파일 열기
2. 모든 코드 복사 (Ctrl+A → Ctrl+C)
3. VBA 에디터 → Insert > Module
4. 빈 모듈에 붙여넣기 (Ctrl+V)
5. 저장 (Ctrl+S)

**코드 2: 견적서생성**
1. GitHub에서 `견적서생성.bas` 파일 열기
2. 모든 코드 복사 (Ctrl+A → Ctrl+C)
3. VBA 에디터 → Insert > Module
4. 빈 모듈에 붙여넣기 (Ctrl+V)
5. 저장 (Ctrl+S)

### 4-3. VBA 에디터 닫기
```
Alt + Q 또는 창 닫기 버튼
```

## 📊 5단계: 각 시트에 데이터 입력

### 5-1. 부품DB 시트 설정

**A1:F1 헤더 입력:**
```
A1: 부품번호
B1: 부품명
C1: 카테고리
D1: 원가
E1: 단위
F1: 규격
```

**A2부터 부품 데이터 입력 예시:**
```
A2: P001          B2: 볼 베어링    C2: 기계부품    D2: 5000
A3: P002          B3: 모터         C3: 전기부품    D3: 25000
A4: P003          B4: 알루미늄봉   C4: 재료        D4: 3000
```

### 5-2. 마진설정 시트 설정

**A1:B1 헤더:**
```
A1: 항목
B1: 마진율
```

**데이터 입력:**
```
A2: 전체_기본_마진     B2: 0.2      (모든 부품 기본값: 20%)

A3: 기계부품           B3: 0.15     (15% 마진)
A4: 전기부품           B4: 0.25     (25% 마진)
A5: 재료               B5: 0.30     (30% 마진)
```

> **팁:** 마진율은 소수점으로 입력 (0.2 = 20%, 0.25 = 25%)

### 5-3. 견적서_작성 시트 설정

**A1:H1 헤더:**
```
A1: 순번
B1: 부품번호
C1: 부품명
D1: 원가
E1: 카테고리
F1: 수량
G1: 판매가
H1: 이익
```

> **주의:** A2부터 데이터 입력 시작

## 🚀 사용 시작!

### 견적서 만드는 법

1. **견적서_작성 시트** 열기

2. **부품 추가:**
   ```
   B2 셀에 부품번호 입력 (예: P001)
   ↓ 엔터 누르기
   ↓ 자동으로 부품명, 원가, 카테고리 조회됨
   ```

3. **수량 입력:**
   ```
   F2 셀에 수량 입력 (예: 10)
   ↓ 엔터 누르기
   ↓ 자동으로 판매가, 이익 계산됨
   ```

4. **부품 반복 추가:**
   ```
   B3, F3, B4, F4... 계속 입력
   ```

5. **견적서 생성:**
   ```
   모든 부품 입력 완료
   ↓
   "견적서 생성" 버튼 클릭
   ↓
   자동으로 새 Excel 파일 생성 & 저장
   ```

6. **생성된 견적서 확인:**
   ```
   Quotation_Tool.xlsm이 있는 폴더
   ↓ Quotations 폴더
   ↓ QT-yyyymmdd-hhmm.xlsx 파일 확인
   ```

## 🆘 문제 해결

### Q1. 부품정보가 자동으로 조회되지 않음
**원인:** 
- 부품DB 시트명이 다름
- 부품 데이터가 없음

**해결:**
- 부품DB 시트명 확인
- 부품DB에 데이터 입력 확인

---

### Q2. 판매가가 계산되지 않음
**원인:** 마진설정이 되어있지 않음

**해결:**
- 마진설정 시트 B2 셀에 값 입력 (예: 0.2)

---

### Q3. VBA 코드 실행이 안 됨
**원인:** 매크로가 비활성화됨

**해결:**
- 파일 > 옵션 > 보안 센터에서 매크로 활성화

---

### Q4. "찾을 수 없음" 오류 메시지 표시
**원인:** 입력한 부품번호가 부품DB에 없음

**해결:**
- 부품DB의 부품번호를 정확하게 확인 후 입력
