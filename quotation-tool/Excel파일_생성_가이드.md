# Excel 파일 생성 및 VBA 설정 가이드

## 📝 Quotation_Tool.xlsm 만드는 방법

### 1단계: Excel 파일 생성

1. **Microsoft Excel 열기**
   - Windows: Excel 실행
   - Mac: Microsoft Excel 실행

2. **새 파일 생성**
   - **파일 > 새로 만들기** 또는 **Ctrl+N**
   - 빈 통합 문서 선택

3. **파일명으로 저장**
   - **파일 > 다른 이름으로 저장**
   - 파일명: `Quotation_Tool`
   - 파일 형식: **Excel 매크로 사용 통합 문서 (.xlsm)** ⭐ (중요!)
   - 저장 위치: 원하는 폴더 선택

---

## 📌 파일 형식 주의

| 확장자 | 이름 | 매크로 가능 여부 |
|--------|------|-----------------|
| .xlsx | Excel 통합 문서 | ❌ 아니오 (사용 불가) |
| **xlsm** | **Excel 매크로 사용 통합 문서** | ✅ **YES (필수!)** |
| .xls | Excel 97-2003 통합 문서 | ⚠️ 옛날 형식 |

> **반드시 .xlsm 형식으로 저장하세요!**

## 🔧 VBA 코드 추가하기

### 2단계: VBA 에디터 열기

1. Excel 파일이 열려있는 상태에서 **Alt + F11** 누르기
   - 또는 **개발 도구 탭 > Visual Basic** 클릭

2. **VBA 에디터** 창이 열림

### 3단계: 모듈 추가

1. **Insert > Module** 클릭
2. 빈 모듈이 생성됨

### 4단계: VBA 코드 복사

#### 코드 1: 메인_매크로.bas
1. GitHub에서 `메인_매크로.bas` 파일 열기
2. 모든 코드 복사 (Ctrl+A → Ctrl+C)
3. VBA 에디터의 Module1에 붙여넣기 (Ctrl+V)
4. 저장 (Ctrl+S)

#### 코드 2: 견적서생성.bas
1. **Insert > Module** 클릭해서 새 모듈 추가
2. GitHub에서 `견적서생성.bas` 파일 열기
3. 모든 코드 복사 (Ctrl+A → Ctrl+C)
4. VBA 에디터의 Module2에 붙여넣기 (Ctrl+V)
5. 저장 (Ctrl+S)

## 📋 필수 시트(Sheet) 생성

VBA 코드가 참조하는 시트 이름이 정확해야 합니다!

### 시트 생성 방법

1. Excel 파일 하단의 시트 탭 우클릭
2. **"시트 삽입"** 또는 **"Insert Sheet"** 선택
3. 다음 이름으로 4개 시트 생성:

| 순서 | 시트명 | 용도 |
|------|--------|------|
| 1 | `부품DB` | 부품 데이터베이스 |
| 2 | `마진설정` | 마진율 설정 |
| 3 | `견적서_작성` | 견적서 작성 (메인 사용 시트) |
| 4 | `견적서_템플릿` | 견적서 템플릿 (선택사항) |

> ⚠️ **시트 이름을 정확하게 입력하세요!** (공백 포함)

---

## 📊 각 시트 구조 설정

### 1. 부품DB 시트

**A1 행(헤더):**
```
A1: 부품번호
B1: 부품명
C1: 카테고리
D1: 원가
E1: 단위
F1: 규격
```

**데이터 예시 (A2부터):**
```
부품번호 | 부품명        | 카테고리   | 원가     | 단위 | 규격
P001    | 볼 베어링     | 기계부품   | 5000    | 개  | 25x52x15mm
P002    | 모터         | 전기부품   | 25000   | 개  | 2HP
P003    | 알루미늄봉   | 재료       | 3000    | m   | Ø20
```

### 2. 마진설정 시트

**A1:B1(헤더):**
```
A1: 항목
B1: 마진율
```

**데이터 (A2부터):**
```
A2: 전체_기본_마진     B2: 0.2    (20%)
A3: 기계부품           B3: 0.15   (15%)
A4: 전기부품           B4: 0.25   (25%)
A5: 재료               B5: 0.30   (30%)
```

### 3. 견적서_작성 시트

**A1 행(헤더):**
```
A1: 순번
B1: 부품번호
C1: 부품명
D1: 원가
E1: 카테고리
F1: 수량
G1: 판매가
H1: 이익
```

**셀 형식:**
```
A: 자동 번호 (1, 2, 3...)
B: 입력 필수 (부품번호 입력)
C: 자동 조회 (B 입력 시)
D: 자동 조회 (B 입력 시)
E: 자동 조회 (B 입력 시)
F: 입력 필수 (수량 입력)
G: 자동 계산 (F 입력 시)
H: 자동 계산 (F 입력 시)
```

## 🎯 매크로 활성화 설정

### Excel 시작 시 보안 경고 표시

1. 파일을 열 때 **"보안 경고"** 나타날 수 있음
2. **"매크로 활성화"** 또는 **"Enable Macros"** 클릭
3. 필요시 **"이 파일을 신뢰"** 선택

### 매크로 비활성화 경우 수동 활성화

1. **파일 > 옵션** (또는 **Tools > Options**)
2. **보안 센터 > 보안 센터 설정**
3. **매크로 설정 > 모든 매크로 활성화** 선택
4. Excel 재시작

## 🚀 사용 시작

모든 설정이 완료되면:

1. **견적서_작성** 시트 열기
2. B2 셀에 부품번호 입력 → 엔터
3. 자동으로 부품명, 원가, 카테고리 조회됨
4. F2 셀에 수량 입력 → 엔터
5. 자동으로 판매가, 이익 계산됨
6. 부품 추가 반복
7. 모든 입력 완료 후 "견적서 생성" 버튼 클릭

## 🔍 트러블슈팅

### 문제: 부품정보 자동 조회가 안 됨
**원인:** 부품DB 시트명이 틀렸거나 부품번호가 없음
**해결:** 
- 부품DB 시트명 확인 ("부품DB" 정확하게)
- 부품 데이터 입력 확인

### 문제: 판매가가 계산되지 않음
**원인:** 마진설정이 안 되어 있음
**해결:**
- 마진설정 시트 B2 셀에 값 입력 (예: 0.2)
- 카테고리별 마진율도 입력

### 문제: VBA 코드가 실행 안 됨
**원인:** 매크로가 비활성화됨
**해결:**
- 보안 경고에서 "매크로 활성화" 클릭
- 또는 파일 > 옵션 > 보안 센터에서 설정 변경
