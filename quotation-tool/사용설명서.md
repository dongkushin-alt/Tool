## 설치 및 초기 설정

### 1단계: 파일 다운로드
- `Quotation_Tool.xlsm` 파일을 다운로드합니다
- Windows 또는 Mac의 원하는 위치에 저장합니다

### 2단계: 매크로 활성화
1. Excel에서 파일 열기
2. "보안 경고" 메시지가 표시되면 **"매크로 활성화"** 클릭
3. 매크로가 비활성화된 경우:
   - **파일 > 옵션 > 보안 센터 > 보안 센터 설정**
   - **매크로 설정 > 모든 매크로 활성화** 선택
   - 파일 다시 열기

### 3단계: 초기 데이터 입력
1. **"부품DB"** 시트 열��
2. 샘플 데이터를 참고하여 부품 정보 입력
3. **"마진설정"** 시트에서 기본 마진율 설정

## 부품 데이터베이스 관리

### 부품DB 시트 구조

| 컬럼 | 항목 | 설명 | 예시 |
|------|------|------|------|
| A | 부품번호 | 고유 부품 코드 (필수) | P001 |
| B | 부품명 | 부품의 정식 명칭 | 볼 베어링 6205 |
| C | 카테고리 | 부품 분류 (마진율 적용용) | 기계부품 |
| D | 원가 | 원가 (세금 제외) | 5000 |
| E | 단위 | 판매 단위 | 개 |
| F | 규격 | 부품 규격 정보 | 25x52x15mm |

### 부품 추가 방법

#### 방법 1: 수동 입력
1. "부품DB" 시트에서 빈 행 선택
2. 각 항목 입력
3. 엔터 키로 저장

#### 방법 2: 기존 Excel에서 복사
1. 기존 부품 목록을 A:F 열로 정렬
2. 데이터 복사
3. "부품DB" 시트의 A2 셀에 붙여넣기 (Ctrl+V)

#### 방법 3: CSV 임포트
1. 부품 데이터를 CSV로 저장
2. **"파일 > 열기"**에서 CSV 파일 선택
3. 데이터 → "텍스트를 열" 선택
4. 데이터 형식 선택 후 완료
5. 결과를 복사하여 "부품DB"에 붙여넣기

### 부품 삭제 및 수정
- 행 전체 삭제: 행 번호 우클릭 → "삭제"
- 부품 정보 수정: 해당 셀 선택 → 수정 후 엔터
- ⚠️ **주의**: 부품DB를 삭제 후 정기적으로 백업하세요

## 마진율 설정

### 마진설정 시트 구조

| 행 | 항목 | 설명 | 입력값 |
|----|------|------|--------|
| 2 | 전체 기본 마진 | 모든 부품의 기본 마진율 | 0.2 (20%) |
| 3 이하 | 카테고리명 | 부품 카테고리 | 기계부품 |
| | 카테고리별 마진율 | 해당 카테고리의 마진율 | 0.15 (15%) |

### 마진율 입력 방법

#### 전체 기본 마진 설정
1. "마진설정" 시트 열기
2. **B2** 셀에 마진율 입력 (소수점 형식)
   - 20% = `0.2`
   - 25% = `0.25`
   - 30% = `0.3`

#### 카테고리별 마진 설정
1. A3 셀부터 카테고리명 입력
   - 예: 기계부품, 전기부품, 재료, 완성품 등

2. B3 셀부터 각 카테고리의 마진율 입력
   ```
   A3: 기계부품    B3: 0.15  (15%)
   A4: 전기부품    B4: 0.25  (25%)
   A5: 재료        B5: 0.30  (30%)
   ```

### 마진율 계산 방법
```
판매가 = (원가 + 원가 × 마진율) × 수량

예시) 원가 10,000원, 마진율 20%, 수량 5개
판매가 = (10,000 + 10,000 × 0.2) × 5 = 60,000원
```

## 견적서 작성

### 견적서_작성 시트 사용법

1. **"견적서_작성"** 시트 열기

2. **부품번호 입력** (B열)
   - 부품DB의 부품번호 입력
   - 엔터 키 누름
   - 자동으로 다음 정보 조회:
     - 부품명
     - 원가
     - 카테고리

3. **수량 입력** (F열)
   - 수량 입력
   - 엔터 키 누름
   - 자동으로 판매가 계산

4. **부품 추가**
   - 필요한 만큼 반복
   - 최대 100개 부품까지 추가 가능

### 견적서_작성 시트 컬럼 설명

| 컬럼 | 항목 | 입력여부 | 설명 |
|------|------|---------|------|
| A | 순번 | 자동 | 1, 2, 3... |
| B | 부품번호 | **입력 필수** | 부품DB의 부품번호 |
| C | 부품명 | 자동 조회 | 부품번호 입력 시 자동 채우기 |
| D | 원가 | 자동 조회 | 부품번호 입력 시 자동 채우기 |
| E | 카테고리 | 자동 조회 | 부품번호 입력 시 자동 채우기 |
| F | 수량 | **입력 필수** | 판매 수량 |
| G | 판매가 | 자동 계산 | (원가 × 마진율) × 수량 |
| H | 이익 | 자동 계산 | 판매가 - (원가 × 수량) |

### 예시: 견적서 작성 절차

```
1단계: B2 셀에 "P001" 입력 → 엔터
      → C2, D2, E2에 자동으로 채워짐

2단계: F2 셀에 "10" 입력 → 엔터
      → G2에 판매가 자동 계산

3단계: B3, F3, B4, F4... 반복

4단계: 모든 부품 입력 완료 후 "견적서 생성" 버튼 클릭
```

## 견적서 생성

### 견적서 생성 방법

1. 모든 부품 정보 입력 완료
2. 견적서_작성 시트에서 **"견적서 생성"** 버튼 클릭
3. 확인 메시지에서 **"예"** 선택
4. 자동으로 새 Excel 파일 생성 및 저장

### 생성된 견적서 구조

생성된 견적서에는 다음 정보가 포함됩니다:

```
┌─────────────────────────────────────┐
│   회사명 / 로고 (필요시 추가)         │
└─────────────────────────────────────┘

견적번호: QT-20260610-0001
견적일자: 2026.06.10
유효기간: 2026.07.10

┌─────────┬────────┬────────┬────┬───────┬────────┐
│부품번호 │부품명  │원가    │수량│판매가 │이익    │
├─────────┼────────┼────────┼────┼───────┼────────┤
│P001    │베어링 │5,000  │10 │60,000│10,000 │
│P002    │모터   │25,000 │5  │150,000│0     │
├─────────┼────────┼────────┼────┼───────┼────────┤
│합계                              │210,000│
└─────────┴────────┴────────┴────┴───────┴────────┘
```

### 견적서 저장 위치

- 기본 저장 위치: `Quotation_Tool.xlsm과 같은 폴더 > Quotations 폴더`
- 파일명: `QT-20260610-1230.xlsx` (자동 생성)
- 예시: `C:\Users\User\Documents\Quotations\QT-20260610-1230.xlsx`

### 견적서 목록 보기

1. Quotation_Tool.xlsm 열기
2. "견적서_작성" 시트의 **"이전 견적서 목록"** 버튼 클릭
3. 생성된 모든 견적서 목록 확인

### 견적서 폴더 열기

1. "견적서_작성" 시트의 **"폴더 열기"** 버튼 클릭
2. Quotations 폴더가 자동으로 열림

## 자주 묻는 질문

### Q1. 부품번호를 입력했는데 자동 조회가 안 됩니다.
**A:** 다음을 확인하세요:
- 부품DB에 해당 부품번호가 등록되어 있는지 확인
- 부품번호의 공백이나 특수문자 확인 (정확히 일치해야 함)
- 매크로가 활성화되어 있는지 확인

### Q2. 수량을 입력했는데 판매가가 계산되지 않습니다.
**A:** 다음을 확인하세요:
- 원가(D열)가 0보다 큰 값인지 확인
- 마진설정 시트에서 마진율이 설정되어 있는지 확인
- 엑셀의 자동 계산 기능이 활성화되어 있는지 확인
  - **수식 > 계산 옵션 > 자동** 선택

### Q3. 부품DB의 여러 항목을 한 번에 추가할 수 있나요?
**A:** 예, 다음 방법으로 가능합니다:
- 기존 부품 목록을 A부터 F 열로 정렬
- 전체 복사 (Ctrl+C)
- 부품DB의 A2 셀에 붙여넣기 (Ctrl+V)
- 또는 CSV 임포트 기능 사용

### Q4. 마진율을 카테고리별로 다르게 설정하고 싶습니다.
**A:** 마진설정 시트에서:
1. A3 이하에 카테고리명 입력
2. B3 이하에 해당 카테고리의 마진율 입력
3. 부품DB에서 카테고리가 정확히 일치해야 적용됨

### Q5. 견적서를 생성했는데 어디에 저장되나요?
**A:** 기본적으로 다음 위치에 저장됩니다:
- `Quotation_Tool.xlsm이 있는 폴더/Quotations/`
- "이전 견적서 목록" 또는 "폴더 열기" 버튼으로 확인 가능

### Q6. 이미 생성된 견적서를 수정하고 싶습니다.
**A:** 다음 방법으로 가능합니다:
1. Quotations 폴더에서 해당 견적서 파일 열기
2. 필요한 부분 수정
3. 파일명을 다르게 하여 저장 (기존 파일 보존)

### Q7. 판매가 공식을 커스터마이징하고 싶습니다.
**A:** VBA 코드 수정이 필요합니다:
1. **Alt + F11** 눌러 VBA 에디터 열기
2. "메인_매크로" 모듈 → `CalculatePrice` 함수
3. 계산 공식 수정:
   ```vba
   ' 현재: CalculatePrice = cost * (1 + marginRate) * quantity
   ' 수정 예: CalculatePrice = cost * (1 + marginRate) + fixedCost
   ```

### Q8. Excel 버전이 2010인데 사용 가능한가요?
**A:** 기본적으로는 가능하지만, 일부 기능(특히 버튼 등)은 동작하지 않을 수 있습니다.
**권장 버전:**
- Windows: Excel 2016 이상
- Mac: Excel 2016 이상

### Q9. 파일이 손상되었을 때 어떻게 하나요?
**A:** 다음 방법을 시도하세요:
1. 파일 이름 뒤에 `.zip` 추가: `Quotation_Tool.xlsm.zip`
2. 압축 풀기
3. `xl/worksheets` 폴더에서 sheet 파일 열기
4. 복구 불가능한 경우, 원본 파일 다시 다운로드

### Q10. 다른 사용자와 파일을 공유하려면?
**A:** 다음을 확인하세요:
1. **파일 > 정보 > 통합 문서 보호** 확인
2. 필요시 암호 설정하여 보호
3. OneDrive 또는 SharePoint에 저장하여 협업 가능
4. 단, VBA 매크로는 신뢰할 수 있는 위치에서만 실행 권장
